Программа быстрой разметки изображений и использованием камеры. Для корректной работы программы необходима USB-камера. Приложение также работает и со встроенными сенсорами.
Для сохранения изображения начните разметку из любого угла. В этот момент данные с камеры перестанут поступать на экран и текущий кадр будет сохранён в папку
Для работы ПО в комплекте идёт OpenCV версии 4.0
